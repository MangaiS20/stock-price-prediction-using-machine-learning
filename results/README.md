# Results

This folder contains the evaluation results of the stock price prediction models developed in this project.

## Models Evaluated

- Long Short-Term Memory (LSTM)
- Random Forest Regression
- XGBoost Regression

## Performance Metrics

| Model | MAE | MSE | RMSE | R² Score |
|------|------:|------:|------:|------:|
| **LSTM** | **₹98.03** | **25,126.50** | **158.51** | **0.9299** |
| Random Forest | ₹232.59 | 84,391.31 | 290.50 | 0.7640 |
| XGBoost | ₹109.22 | 22,479.30 | — | 0.9158 |

## Summary

Among the evaluated models, the **LSTM model** achieved the highest predictive performance with the highest R² Score and the lowest prediction error, demonstrating its effectiveness for stock price forecasting using historical Adani Enterprises data.
